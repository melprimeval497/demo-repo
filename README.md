## 🖱️ Install on Windows

After you visit the release page and download the app, follow these steps:

1. Find the file in your Downloads folder.

2. If the file is a `.zip`, extract it first.

3. If the file is an `.exe`, double-click it to start.

4. If Windows shows a security prompt, choose More info if needed, then choose Run anyway if you trust the source.

5. Wait for the app window to open.

6. If the app does not open, check that the file finished downloading.

If the app does not start, try these steps:

1. Download the file again.
2. Make sure the download finished.
3. Extract the zip file if one was provided.
4. Right-click the app file and choose Open.
5. Restart your PC and try again.
6. Make sure Windows is up to date.
